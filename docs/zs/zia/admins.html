admins¶

The following methods allow for interaction with the ZIA Admin Management API endpoints.

class AdminManagementAPI(api: restfly.session.APISession)¶
add(login_name: str, user_name: str, email: str, **kwargs)¶

Adds a new auditor or admin user.

Parameters

  • login_name (str) – Admin or auditor’s login name. loginName is in email format and uses the domain name associated to the Zscaler account.

  • user_name (str) – Admin or auditor’s username.

  • email (str) – Admin or auditor’s email address.

  • **kwargs – Keyword arguments documented below

Keyword Arguments

comments (str) – Additional information about the admin or auditor.

Returns

The newly created admin or auditor resource record.

Return type

dict
